ePANDDA Endpoint Documentation

publications

Returns specimen occurrence and publication records for a given scientific name. results can be filtered by the optional params

Name Label Type Required? Description
scientific_name text True Taxon to search occurrence records for
journal text False Then name of academic Journal a publication would be found
article text False The name of the journal article the given scientific_name appears in
author text False The name of the author who's article describes the given scientific_name
state_province text False The state/province to search for scientific_name and publication references
county text False The county to search for scientific_name and publication references
locality text False The locality name to search for scientific_name occurences and publication references

occurrences

Returns specimens collected from a given locality

Name Label Type Required? Description
taxon_name text True The taxa to search occurrences for
locality text False The locality name to bound taxonomic occurences to
period text The geologic time period to filter taxon occurrences by
institution_code text The abbreviated institution code that houses the taxon occurrence specimen

geonames

Returns specimens collected from a given locality

Name Label Type Required? Description
geolocation Full Text Geographic Search text False A generic place name, will search across locality, county, state/province and country fields
countryName Country Name text False Full country names
countryCode Country Code (2-Letter) text False 2-letter country code, provided from geonames.org
stateProvinceName State/Province Name text False State or province name. Do not include qualifiers (e.g. State, Province) to return better results
stateProvinceCode State Province Code (2-Letter or 2-Digit) text False State or province name. Do not include qualifiers (e.g. State, Province) to return better results
county County Name text False County name. Do not add "County" to your search string
locality Locality Name text False The specific locality where a specimen was recorded
geoPoint Geographic Point (Longitude, Latitude) text False The Longitude, Latitude to search on. Provide a comma separated string. IMPORTANT geoJSON requries LONGITUDE first.
geoRadius Distance from Geographic Point in Meters text False The distance from the provided geoPoint to search within, in meters

stats

Returns general API statistical information

Name Label Type Required? Description
stateProvinces boolean False The number of unique states represented in the collections
countries boolean False The number of unique countries represented in the collections
counties boolean False The number of unique counties represented in the collections
localities boolean False The number of unique localities represented in the collections
geoPoints boolean False The number of unique geographic coordinates represented in the collections
totalRecords boolean False The count of all specimen/occurrence records included in project from iDigBio and PBDB
taxonomies boolean False The number of unique taxonomic hierarchies represented in the collections
geoPoints boolean False The number of unique geographic coordinates represented in the collections
