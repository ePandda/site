Overview

The API returns JSON with the following information:

Counts
The counts array contains the number of results found in iDigBio, Paleobiology Database and the combined, total number of results. Note these counts are total results and do not take into account the limit parameter.

"counts": {
	"idbCount": 20,
	"pbdbCount": 30,
	"totalCount": 50
}

Criteria
The criteria array contains a summary of the query parameters passed in the API request as well as information about how the term was matched against the database. In the following example, the geonames endpoint was queried with geoPoints and geoRadius. The matchPoints array contains a summary of all matched points within the given radius.
